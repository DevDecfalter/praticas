Sobre

Esta página HTML e o arquivo CSS associado a ela são um exemplo simples que pratica o conceito de modelos de caixas usando HTML5 e CSS3. O HTML contém cinco <div>`s com diferentes classes, cada uma estilizada de maneira distinta no CSS. O CSS define estilos básicos, como cor de fundo, largura, altura, preenchimento, borda, margem e posição flutuante para as <divs>'s. O resultado é uma demonstração visual do modelo de caixas em ação, onde cada <div> é exibida com suas características específicas de layout.

Exercício: Width & Height

Sobre

Esta página HTML e o arquivo CSS associado são um exemplo simples de prática de conceitos de largura (width) e altura (height) em HTML5 e CSS3. O HTML contém diferentes seções, cada uma representando um exercício prático com elementos <div>. Cada exercício tem sua própria aparência definida no CSS, demonstrando diferentes cenários de aplicação para largura e altura, como largura e altura fixas, divs lado a lado e divs com largura e altura dinâmicas. A página possui um fundo preto com texto branco para facilitar a leitura.
